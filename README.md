## Instalação

```bash
$ npm install
```

## Rodar a aplicação

```bash
# iniciar banco de dados e build aplicação
$ cd docker
$ docker-compose up -d

# desenvolvimento
$ npm run start:dev

# produção
$ npm run start:prod

```

## Teste

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
